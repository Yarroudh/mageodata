Développement

Cartes interactives

Cartes interactives générées avec Leaflet JS, bibliothèque Javascript Open Source pour la cartographie Internet.

  • Leaflet fonctionne efficacement sur toutes les plateformes mobiles et desktop.
  • Elle peut être étendu avec de nombreux plugins, possède une puissante API, facile à utiliser et bien documentée.

Visualisation des données

Graphiques simples et flexibles créés avec Chart JS, une bibliothèque Javascript Open Source pour la visualisation de données.

Chart JS permet d'afficher des graphiques en forme de courbes, de barres, de radar, de camembert, de graphe polaire et de beignets, simples à intégrer et compatibles avec tous les navigateurs récents.
